# DealNifty Backend

![Node.js](https://img.shields.io/badge/Node.js-18.0-green?logo=node.js) 
![Express](https://img.shields.io/badge/Express-4.18-lightgrey?logo=express) 
![MongoDB](https://img.shields.io/badge/MongoDB-6.0-brightgreen?logo=mongodb)

**DealNifty Backend** is a RESTful API server built with **Node.js**, **Express**, and **MongoDB**. It powers the DealNifty frontend by managing products, users, and orders efficiently.

---

## Features

- **User Authentication & Authorization**  
  - Sign up, login, and logout  
  - Password forgot functionality
  - Role-based access control (admin/user)  

- **Product Management**  
  - CRUD operations for products  
  - Image upload and storage handling  
  - Search, filter, and category support  

- **Order & Cart Management**  
  - Add/remove products from cart  
  - Place orders and track status  
  - Update stock and handle inventory  

- **Review & Rating System**  
  - Users can submit reviews for products  
  - Admins can manage reviews  

- **Secure API**  
  - JWT-based authentication  
  - Input validation and error handling  

- **Optimized for Performance**  
  - MongoDB queries optimized for speed  
  - Scalable REST architecture  

---

**Live Frontend Integration:** Works seamlessly with the [DealNifty Frontend](https://dealnifty.netlify.app)
